# 💍 結婚式準備アプリ「Roots Customer」のご利用ガイド

この度は結婚式という特別な日の準備をサポートさせていただき、誠にありがとうございます！
このアプリは、いつ・何をすればいいか迷いがちな準備を「LINEひとつ」でスムーズに進められる専用のサポートツールです。

## 📱 1. アプリのはじめ方
1. **公式LINEの登録**
   私たちRootsの公式LINEアカウントを「友だち追加」してください。
2. **準備ダッシュボードを開く**
   LINEのトーク画面下部にあるメニューボタン（またはご案内した初回専用URL）をタップすると、アプリが立ち上がります。
3. **ニックネームの登録（初回のみ）**
   最初に「ニックネーム（お名前）」を入力して「はじめる」ボタンを押してください。以上で設定はすべて完了です！

## ✅ 2. タスク（やることリスト）の進め方
ダッシュボード画面には、おふたりがこれから結婚式までに「やるべきこと」が順番にリストアップされています。

1. **タスクの詳しい内容を確認する**
   気になるタスクのカードをタップすると、そのタスクの「詳しい説明」や「参考マニュアルのリンク（※順次追加予定）」を見ることができます。
2. **タスクを終わらせる**
   実際に作業が終わったら、タスクの横にある「チェックボックス（☐）」をタップしてください。
   タスクに終了のチェックが入り、画面の「完了済み」タブへ自動で移動します！達成感を味わいながら進められます✨
3. **どんどん未完了をゼロに！**
   「未完了」のタブにあるタスクが式当日に向けてゼロになるように、おふたりで協力して進めていきましょう。

## 🤖 3. 分からないことは「LINEで質問」！
「招待状の宛名ってどう書くんだっけ？」「引き出物の相場は？」など、作業中に迷うことがあれば、**このLINEのトーク画面にそのまま質問を送信してください！**

当システムの専属AIアシスタントが、おふたりの現在の「タスクの進み具合」を裏側で把握した上で、最適なアドバイスを24時間いつでも即座にご返答します。
もしAIでも解決できない複雑な内容の場合は、担当プランナーが直接メッセージを引き継ぎますので、どんな些細なことでもお気軽にご相談ください！
